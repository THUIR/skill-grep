# Skill Compass User Agreement

Effective date: 2026-03-28

By accessing or using Skill Compass (the "Service"), you agree to this User Agreement.

## 1) Who We Are

Skill Compass is a search and discovery interface for agent skills and related technical content.

## 2) Scope

This Agreement applies to:

- The Skill Compass website and frontend pages.
- Related backend API interactions used by the website.
- Public repository materials where this Agreement is published.

## 3) What Data We Collect

When you use the Service, we may collect:

- Search interaction data (for example: query text, search parameters, timestamps).
- Click interaction data (for example: which result was clicked, rank position, source page).
- Feedback data you submit (for example: rating signals, selected result IDs, optional comments).
- Session-level technical metadata (for example: session IDs, request metadata needed for operation, debugging, and analytics).

We do not intentionally collect special-category sensitive personal data. Please do not submit sensitive personal information in search or feedback fields.

## 4) Why We Collect Data

We use collected data for:

- Operating, maintaining, and improving the Service.
- Measuring retrieval quality and user experience.
- Security, abuse prevention, and troubleshooting.
- Potential future research use, including building research datasets from service interaction data.

If data is used for research datasets, we aim to apply data minimization and de-identification practices before research release or sharing.

## 5) What We Do Not Do

Unless separately disclosed in writing:

- We do not sell your personal data.
- We do not use interaction data for third-party ad targeting profiles.

## 6) Research Use Notice

You acknowledge that interaction data generated through normal Service use may be analyzed for scientific or academic research purposes.

Where feasible and appropriate, we will:

- Remove or reduce directly identifying elements.
- Restrict internal access to authorized personnel.
- Use aggregated or de-identified forms for publication or sharing.

## 7) Soft-Consent Model

This Service uses a soft-consent notice model:

- By continuing to use the Service after being notified of this Agreement, you agree to data processing described here.
- Links to this Agreement are provided in visible website locations and repository documentation.

If you do not agree, please stop using the Service.

## 8) User Choices and Requests

You may request, subject to applicable law and operational limits:

- Information about your data usage in this Service.
- Correction or deletion of data reasonably linked to you.
- Opt-out handling for future telemetry collection where supported.

## 9) Data Retention

We retain data only as long as reasonably needed for service operation, security, quality evaluation, and research purposes, unless a longer period is required by law.

Retention windows may be updated as the project matures and governance requirements evolve.

## 10) Security

We apply reasonable technical and organizational safeguards, but no system is perfectly secure. You use the Service at your own risk.

## 11) Changes to This Agreement

We may update this Agreement periodically.

- The "Effective date" above indicates the current version date.
- Material updates will be posted in the repository and/or website entry points.

Continued use of the Service after updates means you accept the revised Agreement.
